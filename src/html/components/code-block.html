Add a class like language-php for syntax highlighting powered by Prism.js.

Note that Prism forces you to use the correct element for marking up code: <code>. On its own for inline code, or inside a <pre> for blocks of code. See example below.

<pre><code class="language-html">
	// Sample code...
</code></pre>

Prism supports HTML, PHP, JS, CSS, SCSS and many other languages.
