Juniper is a content focused web style guide and component library for use on projects under the Human Made brand. It can be used as a reference, or you can include the files in your project and hit the ground running, allowing you to focus on building the exciting stuff.

Features

  • Base styles for typography and standard HTML elements.
  • Brand elements such as logos & colour swatches.
  • Commonly used components (e.g. navigation).
  • Form and button styles.
  • WordPress styles.

Usage Instructions

To get started, just download and include the production files in your project. See detailed usage instructions

Contributing

We are always looking to improve the Juniper project. New components will be added and we are keen to improve the codebase over time. We will not be committed to backwards compatability between major releases.

If you build something with Juniper, and create a component that may be useful to others, we'd love to include it! Open an issue on Github
